가상환경 생성 : python -m venv dataA

가상환경 활성화 : dataA\Scripts\activate

가상환경 비활성화 : deactivate

가상환경 폴더 삭제 : rmdir /s /q dataA

파이썬 모듈 설치(통으로) : pip install -r python_modules.txt

파이썬 파일 build : ctrl + shift + p
                        interpreter 선택
                        f5 or ctrl + f5
                        종료 시 : shift + f5
